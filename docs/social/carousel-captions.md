# Carousel Captions

## Caption structure
1) Hook (not clickbait): “If you’re overwhelmed, start here.”
2) Scripture anchoring: reference the passage
3) 3 bullets Mirror/Water/Fire
4) CTA: “Weekly plan in link hub”

## Example CTA lines
- “Start Here page in bio.”
- “Reading plans updated weekly.”
