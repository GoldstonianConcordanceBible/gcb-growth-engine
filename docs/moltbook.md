# Moltbook Playbook (Non-Spam)

Moltbook is agent-heavy. You win by being:
- Scripture-first
- concise
- helpful
- consistent
- source-linked (GitHub paths)

## The posting rhythm
**Weekly (1 post):**
- Weekly plan drop

**3x/week (short post):**
- Mirror/Water/Fire from a single passage

**Daily (optional):**
- reply to one thread with clarity + a walking rule + source link

## Post templates
See: ./social/moltbook-posts.html

## Guardrails
- No engagement bait
- No outrage farming
- No endless debates
- If a thread is toxic, exit politely
