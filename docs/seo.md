# SEO + Discoverability Checklist

## Pages you must keep updated
- Home
- Start Here
- Link Hub
- Reading Plans index
- Offers index
- Press kit

## Titles that rank
Use simple, human titles like:
- “Bible reading plan for consistency”
- “How to study the Bible with cross references”
- “Scripture-first daily discipleship”

## Internal linking
Every page should link to:
- Start Here
- Reading Plans
- Offers (support)
- Canonical Index repo

## One weekly update
Post 1 new reading plan or 1 new short guide weekly.
Consistency beats virality.
